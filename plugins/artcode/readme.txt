== Description ==
Create ArtCode experience from wordpress content (pages and posts). 

ArtCodes, http://aestheticodes.com/, are aestically pleasing machine-readable codes. This plugin allows you to create ArtCode experiences for use with the ArtCode App out of posts and pages in wordpress. 

== Installation ==
1. Upload the directory `artcode` to the `/wp-content/plugins/` directory
1. Activate the plugin through the \'Plugins\' menu in WordPress
